# Catálogo Builder — Villalba Hnos.

Herramienta profesional para la creación de catálogos de productos médicos y cajas instrumentales. Diseñada específicamente para el ADN visual de Villalba Hnos.

## 🚀 Características

- **Diseño Premium**: Interfaz moderna con animaciones y micro-interacciones.
- **Exportación PDF**: Generación de documentos A4 con formato de imprenta exacto.
- **Persistencia en la Nube**: Integración completa con Firebase Firestore para guardar y cargar proyectos.
- **Editor Visual**: Posicionamiento dinámico de callouts y escalado de imágenes en tiempo real.
- **Responsive**: Optimizado para flujos de trabajo eficientes.

## 🛠️ Tecnologías

- **HTML5 / CSS3 (Vanilla)**
- **JavaScript (ES6+)**
- **Firebase** (Firestore & Analytics)
- **html2pdf.js** (Captura y generación de PDF)

## 📦 Configuración

Para usar tu propia base de datos, actualiza el archivo `app.js` con tus credenciales de Firebase:

```javascript
const firebaseConfig = {
  apiKey: "TU_API_KEY",
  authDomain: "TU_PROYECTO.firebaseapp.com",
  // ...
};
```
